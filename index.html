Wi by mockko

5 CSS/JS widgets for gorgeous, native-quality iPhone web apps

5 widgets (backgrounds, bars, buttons, segmented control, slider) for iPhone/iPad/iPod Touch WebKit browsers. Native quality. No dependencies. Hardware acceleration. Commercial-friendly open-source MIT license. All artwork recreated in Photoshop specifically for Wi. * Wi.Backgrounds: a growing set of standard iPhone backgrounds. More cool background soon. * Wi.Bars: navigation bar, toolbar, search bar, browser bar, custom bars, tinted bars. * Wi.Buttons: a collection of pre-styled configurable sexy iPhone buttons, pick your own color or choose one of the stock colors * Wi.Segmented: UISegmentedControl meets Web * Wi.Slider: iPhone slider These widgets are meant for general use in iPhone web app development, but they are also employed in (and were created for) the Mockko iPhone Designer application (http://www.mockko.com/).

Dependencies

None.

Install

Just drop `wi.css` and `wi.js` into your app (best for development), or include `widgets/?/?.css` and `widgets/?/?.js` only for the widgets you actually use (best for production). Minified versions will be provided soon.
